# المساهمة في B-Chat 🤝

شكراً لاهتمامك بالمساهمة! إليك كيفية المساهمة:

## 📋 خطوات المساهمة

1. **Fork المستودع**
   - اذهب إلى https://github.com/01145463698/b-chat
   - اضغط زر "Fork"

2. **استنسخ المستودع الخاص بك**
   ```bash
   git clone https://github.com/YOUR_USERNAME/b-chat.git
   cd b-chat
   ```

3. **أنشئ فرع جديد**
   ```bash
   git checkout -b feature/اسم-الميزة
   ```

4. **قم بالتغييرات**
   - عدّل الملفات المطلوبة
   - تأكد من الاختبارات

5. **أضف التغييرات**
   ```bash
   git add .
   git commit -m "وصف التغيير"
   ```

6. **ادفع إلى GitHub**
   ```bash
   git push origin feature/اسم-الميزة
   ```

7. **أنشئ Pull Request**
   - اذهب إلى مستودعك
   - اضغط "New Pull Request"
   - اكتب وصفاً واضحاً

## 📝 معايير الكود

- اكتب أكواداً نظيفة وقابلة للقراءة
- أضف تعليقات للأجزاء المعقدة
- اتبع نفط نمط الكود الموجود
- اختبر كودك قبل الإرسال

## 🐛 الإبلاغ عن الأخطاء

- استخدم "Issues" في GitHub
- صف المشكلة بوضوح
- أرفق لقطات شاشة إن أمكن

## 🙏 شكراً!

كل مساهمة تقربنا من تطبيق أفضل!
